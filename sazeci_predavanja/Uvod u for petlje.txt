Lecture: "Uvod u for petlje" (Intro to for Loops)
7 March 2026 — 1h 56m — Edukacijska grupa Ada

================================================================================
TOPICS COVERED
================================================================================

1. C++ Syntax Warmup (pre-loop)
   - Variable increment: a = a + 1
   - ++ operator (pre vs post-increment): a++ vs ++a
     Post-increment returns the old value; pre-increment returns the new value
   - Compound assignment operators: +=, -=, *=, /=, %=

2. for Loops
   - Motivation: printing "hello" 100 times without copy-pasting
   - Syntax: 3 components — init (int i = 1), condition (i <= 100), increment (i++)
   - Controlling behavior: changing start value, step size (i += 2), and condition
   - Modifying the loop variable inside the body (e.g., setting i = 5 mid-loop causes early exit)

3. while Loops
   - Motivation: when iteration count is unknown in advance
   - Syntax: while (condition) { ... }
   - When to use for vs while: for when count is known, while when stopping condition is dynamic

4. Accumulator Pattern
   - Running sum (init at 0) and running product (init at 1, not 0)

          | Key concept: while loop motivation
